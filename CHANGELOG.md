# История изменений

## [1.0.0] - 2026-02-03

### Добавлено
- ✅ Полная реализация серверной части (ASP.NET Core Web API)
- ✅ Полная реализация клиентского приложения (WPF)
- ✅ Система аутентификации и авторизации (JWT)
- ✅ Управление пользователями (регистрация, вход)
- ✅ CRUD операции для задач
- ✅ Управление категориями
- ✅ Система тегов
- ✅ Фильтрация и поиск задач
- ✅ Статистика выполнения
- ✅ Material Design интерфейс
- ✅ Swagger документация API
- ✅ База данных SQLite
- ✅ Entity Framework Core
- ✅ MVVM архитектура в WPF
- ✅ Безопасное хранение паролей (BCrypt)

### Функционал
- Приоритеты задач (Низкий, Средний, Высокий, Критический)
- Статусы задач (Новая, В процессе, Выполнена, Отложена, Отменена)
- Категории с цветовым кодированием
- Установка сроков выполнения
- Отслеживание просроченных задач
- Поиск по названию и описанию
- Фильтрация по статусу и категории

### Документация
- ✅ Техническое задание (ГОСТ 7.32–2017, ГОСТ Р 7.0.5–2008)
- ✅ README с полной документацией
- ✅ Инструкция по быстрому старту
- ✅ Описание API endpoints
- ✅ Схема базы данных

### Технические детали
- .NET 6.0
- ASP.NET Core Web API
- WPF с Material Design
- Entity Framework Core
- SQLite
- JWT Bearer Authentication
- BCrypt для паролей
- Swagger/OpenAPI
